> A react(no CRA)/Express/mongoDB Othello game

## Prerequisites

- npm 6.13.6
- node 13.8.0
- express ^4.17.1
- reversi ^3.0.0
- mongoose ^5.8.9
- react ^16.12.0
- @babel/runtime ^7.2.0
- webpack ^4.41.5

## Install

```sh
npm install
```

## Launch client

```sh
npm run webpack
```

## Launch server

```sh
npm run server
```

## Run client tests

```sh
npm run test:client
```
## Run linter

```sh
npm run lint
```
